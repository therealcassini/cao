控制台调试日志查看指南

步骤1：打开浏览器开发者工具

在知识图谱页面（http://localhost:5173/），按以下快捷键打开开发者工具：

Windows/Linux: Ctrl + Shift + I
Mac: Cmd + Opt + I

步骤2：查看控制台日志

在开发者工具中，切换到"Console"（控制台）标签页，查看详细的边处理日志信息。

您应该能看到类似以下的日志输出：

===== 原始边数据详情 =====
边1: ID=1, 源节点="韩立", 目标节点="南宫婉", 关系="夫妻"
边2: ID=2, 源节点="韩立", 目标节点="厉飞雨", 关系="好友"
...
==========================

边ID: 1, 源节点: "韩立" -> ID=1, 目标节点: "南宫婉" -> ID=2
边ID: 2, 源节点: "韩立" -> ID=1, 目标节点: "厉飞雨" -> ID=3
...

===== 边处理统计 =====
总边数: 3
有效边数: 3
无效边数: 0
=====================

处理后的边数据: [ {source: 1, target: 2, name: "夫妻"}, ... ]

步骤3：验证解决方案

如果所有边都显示为有效边，并且处理后的边数据不为空，那么边应该能正确显示在知识图谱中。
